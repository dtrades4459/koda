Smoke-test runbook — copy/paste-able

Two-hour focused block. Sequence below assumes the Vercel preview is live; replace PREVIEW with the actual URL. Order matters: do A → B → C → D in sequence so one failure tells you exactly what's broken.

A · Smoke the preview URL itself (2 min)
  • Open PREVIEW in your phone's browser. Page loads, fonts render, no white screen.
  • If the deploy failed: check Vercel Dashboard → Deployments → latest → Build Logs. Tell me the error.
B · Founder dashboard (3 min)
  • Sign in on the preview with dnyland420@gmail.com.
  • Navigate to PREVIEW/admin. The dashboard should load with WAJ / signups / activation / runway.
  • If 403 "Forbidden": your ADMIN_EMAILS env var doesn't match your sign-in email (or env var isn't on the preview scope).
  • If 503 "Admin allowlist not configured": ADMIN_EMAILS wasn't set or the deploy is older than the env-var add. Trigger a fresh deploy.
C · Email round-trips (15 min)

Use a real inbox you own. Some emails take 10–60 s to arrive (Resend free tier).

  • Welcome — Sign up a fresh test account on the preview, finish onboarding. Email should land within ~30 s, from noreply@kodatrade.co.uk, with the cat12 design (mint accent, italic verb).
  • Email verification — Same signup flow. Branded verification email arrives instead of Supabase's default (because you suppressed it).
  • Password reset — Sign-in screen → "Forgot password?" on the test account. Needs a recovery_email in user metadata.
  • Beta-unlock — On the beta gate, enter your beta code with an email. Confirmation email arrives.
  • Account deletion — Settings → Delete account. Confirmation email lands with the 14-day grace date. Sign back in → click "Keep my account" → verify deletion_scheduled_for is cleared in user_kv.
  • Payment failed — stripe trigger invoice.payment_failed
  • Subscription cancelled — stripe trigger customer.subscription.deleted
D · Crons (5 min)

Manually fire each so you don't have to wait for the schedule. CRON_SECRET is in Vercel env.

curl -H "Authorization: Bearer $CRON_SECRET" PREVIEW/api/cron?job=streak-milestones
curl -H "Authorization: Bearer $CRON_SECRET" PREVIEW/api/cron?job=monthly-summary
curl -H "Authorization: Bearer $CRON_SECRET" PREVIEW/api/cron?job=waitlist-positions
curl -H "Authorization: Bearer $CRON_SECRET" PREVIEW/api/cron?job=delete-expired-accounts

Each returns {"ok":true,…}. Most will report sent: 0 on a fresh DB — that's expected. Check Vercel function logs if any return errors.

E · PWA on Android (5 min, skip if iOS-only)
  • Install the PWA from preview URL (Chrome menu → Install).
  • Long-press the home icon → 4 shortcuts appear (Log / Stats / Circles / Import).
  • From any app, share a screenshot → "Kōda" → lands on share-receive view with the image previewed.
  • OS shelf badge mirrors the bell-icon count when there are unread drafts.
F · Marketing pages (1 min)
  • Open each in sequence: PREVIEW/about.html, /blog.html, /contact.html, /press.html. Visual scan only — fonts, layout, no broken links.
